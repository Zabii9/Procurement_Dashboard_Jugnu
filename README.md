# Procurement Intelligence Dashboard

An advanced analytics and intelligence dashboard for procurement teams, built with Streamlit and Plotly.

## 🌟 Features
- **Real-time KPIs**: Monitor purchase value, inventory health, and OOS rates.
- **Supplier Analytics**: Evaluate manufacturer performance and risk.
- **Inventory Management**: Track coverage, stock status, and demand gaps.
- **Target Tracking**: Compare MTD performance against SKU-level targets.
- **Working Capital Analysis**: Optimize cash flow and Days of Cover (DOC).

## 📄 Documentation
For detailed instructions on how to use each tab and understand the metrics, please refer to the **[User Guide](user_guide.md)**.

## 🚀 Getting Started
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:
   ```bash
   streamlit run streamlit_app.py
   ```
